SWIB 2020 - Sinopia Linked Data Editor

Editor

While designed as an generic Linked-Data editor, the Sinopia editor and supporting stack is optimized for the BIBFRAME bibliographic vocabulary developed by the Library of Congress.

With RDF templates being developed by PCC for BIBFRAME Works, Instances, and Items, along with their sub-classes, catalogers describe the resource and can mint URIs for each of these classes and link them through Sinopia's interface.

Depending on how the RDF template creator structured the RDF template, Sinopia allows catalogers to link external entities into a constructured triple using an external look-up service called Questioning Authority (QA) or directly from the Library of Congress.

BIBFRAME Workflows

With a focus on native linked-data cataloging using the BIBFRAME vocabulary, Sinopia allows catalogers to open up multiple, related templates with tabs to enable quick creation of related Work-Instance-Item entities, each with a minted URI.
